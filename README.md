## 🚀 Getting Started 启动

#### 1. Clone and Install

```bash
# Clone the repo
https://github.com/fancn21th/0_35_React_Native_Todo_List

# Install dependencies
npm i
```

#### 2.1. Run the _React Native_ App on Mac

```bash
# Start the React Native App
npm run ios
```
